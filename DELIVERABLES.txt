ENTITIES & DATABASE
═══════════════════════════════════════════════════════════════════════════════

Used Entities (All pre-existing):
1. AgentProfile - Main agent entity with caching fields
2. AgentPlatformMetrics - Monthly KPIs per platform
3. AgentHistoricalPerformance - Yearly trend data
4. AgentPerformanceSyncLog - Sync audit trail
5. ServiceEngagement - Agent to workspace mapping
6. ConnectedAccount - OAuth credentials storage

Database Tables:
1. agent_platform_metrics
   - Monthly rollups of KPIs per platform
   - Indexed on (agent_profile_id, platform, aggregation_period)
   - Stores: spend, campaigns, ROAS, CPA, CTR, conversions, revenue

2. agent_historical_performance
   - Yearly trends with success rates
   - Unique constraint on (agent_profile_id, year_month)
   - Stores: campaign counts, spend, KPIs, best ROAS

3. agent_performance_sync_logs
   - Audit trail of all sync operations
   - Tracks: status, records, errors, timing, retry schedule

4. agent_profiles (updates)
   - cachedStats: Cached JSONB with aggregated KPIs
   - lastPerformanceSync: Timestamp of last sync
   - performanceSyncStatus: health|stale|failed|never_synced

SERVICE METHODS
═══════════════════════════════════════════════════════════════════════════════

PUBLIC API (4 methods):
1. syncAgentPerformance(agentId, options?) → Promise<SyncResult>
   - Main orchestration method
   - Supports optional platform filtering
   - Returns comprehensive sync result with timing

2. getSyncHistory(agentId, limit?) → Promise<AgentPerformanceSyncLog[]>
   - Get recent sync logs for monitoring
   - Pagination support
   - Includes error tracking

3. getCurrentMetrics(agentId) → Promise<AgentPlatformMetrics[]>
   - Get current month KPIs
   - Per-platform breakdown
   - Ready for marketplace display

4. getHistoricalPerformance(agentId, year?) → Promise<AgentHistoricalPerformance[]>
   - Get yearly trends
   - Optional year filtering
   - Includes success rates and best metrics

PRIVATE METHODS (7 methods):
1. syncMetaPerformance() - Meta Ads API integration stub
2. syncGooglePerformance() - Google Ads API integration stub
3. syncYandexPerformance() - Yandex Direct API integration stub
4. storePlatformMetrics() - Database persistence layer
5. aggregateAndCacheStats() - KPI calculation and caching
6. logSync() - Audit trail logging
7. calculateNextSyncTime() - Smart retry scheduling

═══════════════════════════════════════════════════════════════════════════════
INTERFACE DEFINITIONS
═══════════════════════════════════════════════════════════════════════════════

SyncOptions:
{
  forceSync?: boolean;                 // Force full re-sync
  platformsToSync?: string[];          // Specific platforms only
}

SyncResult:
{
  success: boolean;                    // Overall success
  agentId: string;                     // Agent UUID
  recordsSynced: number;               // Total campaigns
  platformsProcessed: string[];        // Platforms that synced
  metricsStored: number;               // Metrics rows created
  errors: string[];                    // Error messages
  syncDuration: number;                // Milliseconds elapsed
  lastSyncedAt: Date;                  // Timestamp
}

PlatformMetrics:
{
  platform: string;                    // 'meta'|'google'|'yandex'
  totalSpend: number;                  // Total ad spend
  campaignCount: number;               // Number of campaigns
  conversions: number;                 // Total conversions
  revenue: number;                     // Total revenue
  avgRoas?: number;                    // Return on ad spend
  avgCpa?: number;                     // Cost per acquisition
  avgCtr?: number;                     // Click through rate
}

AggregatedStats:
{
  avgROAS: number;                     // Weighted ROAS
  avgCPA: number;                      // Weighted CPA
  avgCTR: number;                      // Mean CTR
  totalCampaigns: number;              // Sum of campaigns
  activeCampaigns: number;             // Active campaigns count
  successRate: number;                 // Success percentage
  totalSpendManaged: number;           // Total spend all-time
  bestROAS: number;                    // Best ROAS observed
}
