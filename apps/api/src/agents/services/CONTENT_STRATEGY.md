## Meta Tag Best Practices

### Title Tags

**Format:** `[Specialist/Category] [Platform] [Unique Value] | Performa`

**Examples:**
- "John Smith - 5.2x ROAS Meta Ads Expert | Performa"
- "E-commerce Marketing Specialists (180+ Results) | Performa"
- "Find AI Marketing Specialists | Performa Marketplace"

**Requirements:**
- 50-60 characters ideal
- Include primary keyword
- Include unique value proposition (ROAS, certification, location)
- Start with most important keyword
- Include brand name at end

**Do's:**
- Include performance metrics for specialists
- Use numbers and stats
- Mention certifications
- Include geographic location if relevant

**Don'ts:**
- Keyword stuffing
- Misleading claims
- Duplicate titles
- Special characters (unless relevant)

### Meta Descriptions

**Format:** `[Brief description]. [Key attributes: stats/certifications/niche]. [Call to action].`

**Examples:**
- "Hire John Smith - Certified Meta ads expert achieving 5.2x ROAS on e-commerce campaigns. 42 verified reviews. Available for consultation."
- "Browse 180+ certified performance marketing specialists. Filter by platform, niche, location & ratings. Find your perfect expert on Performa."
- "Connect with certified advertising specialists. Proven track record managing ads on Meta, Google, Yandex. Verified ratings & case studies."

**Requirements:**
- 150-160 characters ideal
- Include primary keyword naturally
- Include key benefits or stats
- Include call to action
- Be descriptive and compelling

**Do's:**
- Include specific metrics (ROAS, rating, review count)
- Mention certifications
- Highlight unique benefits
- Include target audience
- Add social proof (reviews, experience)

**Don'ts:**
- Duplicate descriptions
- Keyword stuffing
- Truncated information
- Generic descriptions

## Page-Specific Content Strategy

### Marketplace Landing Page

**Primary Goal:** Drive qualified traffic and establish marketplace authority

**Key Sections & Keywords:**

1. **Hero Section**
   - Main keywords: "AI marketing specialist", "performance marketing"
   - Sub-message: 150+ verified specialists
   - Include trust signals: Average rating, managed spend, client count

2. **Featured Specialists**
   - Top performers by rating/ROAS
   - Diverse specializations and platforms
   - High review counts

3. **How It Works**
   - Simplify specialist selection process
   - Emphasize verification and quality
   - Show conversion path

4. **FAQ Section**
   - Address common search queries
   - Use long-tail keywords naturally
   - Include "people also ask" format

5. **CTA Section**
   - Strong call-to-action
   - Multiple entry points
   - Mobile-optimized buttons

**Expected Ranking Keywords:**
- "find marketing specialist" - Position 1-3
- "hire marketing expert" - Position 3-5
- "marketing specialist marketplace" - Position 1-3

### Specialist Profile Pages

**Primary Goal:** Convert specific searches to consultant engagements

**Key Content Sections:**

1. **Profile Header**
   - Name, title, avatar
   - Rating and review count
   - Key metrics: Avg ROAS, experience level
   - Certifications as badges

2. **Bio/Expertise**
   - 100-150 word summary
   - Primary platforms and specializations
   - Experience level and track record
   - Unique value proposition

3. **Certifications**
   - All verified certifications with issue dates
   - Certification badges with proper alt text
   - Certification links for verification

4. **Performance Metrics**
   - Average ROAS with comparison
   - Total campaigns managed
   - Industries served
   - Success rate

5. **Case Studies**
   - Real examples with metrics
   - Before/after performance
   - Client testimonials
   - Result verification

6. **Reviews Section**
   - Star rating display
   - Recent reviews prominently
   - Review date and reviewer info
   - Search-optimized review keywords

7. **Geographic Coverage**
   - Supported countries
   - Primary markets
   - Remote availability
   - Timezone coverage

8. **Languages**
   - Supported languages
   - Proficiency levels
   - Communication channels

9. **Pricing & Engagement**
   - Clear pricing structure
   - Monthly rate or commission model
   - Minimum budget if applicable
   - Payment terms

**Expected Ranking Keywords:**
- "[Specialist Name] marketing specialist" - Position 1
- "[Platform] ads expert" - Position 3-10
- "[Niche] marketing professional" - Position 3-10

### Search Results Pages

**Primary Goal:** Capture users actively searching for specialists

**Dynamic Keywords Based on Filters:**

1. **Platform Filters**
   - "Meta ads specialist" (42 results)
   - "Google ads expert" (38 results)
   - "Yandex ads professional" (15 results)

2. **Niche Filters**
   - "E-commerce marketing specialist" (28 results)
   - "SaaS marketing expert" (22 results)

3. **Location Filters**
   - "US marketing specialist" (67 results)
   - "UK marketing expert" (34 results)

4. **Combined Filters**
   - "Meta ads specialist in e-commerce" (12 results)
   - "Google ads expert for SaaS near me" (8 results)

**Page Content Optimization:**
- Title includes primary filter + result count
- Description explains filter selection
- Show active filters prominently
- Display available refinement options
- Include sort options (rating, ROAS, price, newest)

## Content Optimization Best Practices

### Heading Hierarchy (H1-H3)

**Marketplace Landing Page:**
- H1: "Find Your Perfect Advertising Expert"
- H2: "Filter by Expertise", "Top-Rated Specialists", "How It Works"
- H3: Individual section details

**Specialist Profile Page:**
- H1: "[Specialist Name] - [Platform] Ads Expert"
- H2: "About", "Expertise", "Performance", "Case Studies", "Reviews"
- H3: Subsection details within each section

**Search Results Page:**
- H1: "[Platform/Niche] Marketing Specialists"
- H2: "Filters", "Results", "Sort Options"
- H3: Individual specialist cards (if needed)

### Image Alt Text

**Specialist Avatar Images:**
- "John Smith - Certified Meta ads specialist"
- "[Specialist Name] - [Platform] ads expert profile picture"

**Case Study Images:**
- "Meta ads campaign results showing 5.2x ROAS increase"
- "Before/after comparison of e-commerce campaign optimization"

**Certification Badges:**
- "Meta Blueprint Certification badge"
- "Google Ads Certified Professional badge"

**Platform Icons:**
- "Meta Ads icon"
- "Google Ads icon"

### Internal Linking Strategy

**From Marketplace Landing Page:**
- Link to top specialists: "Meet our top performers"
- Link to search: "Browse all specialists"
- Link to platform-specific pages: "Meta specialists" (create dynamic pages)

**From Specialist Profile Page:**
- Link to similar specialists: "Other [platform] experts"
- Link to industry category: "More e-commerce specialists"
- Link back to search: "Browse more specialists"

**Cross-Linking:**
- Specialists serving same industry
- Specialists on same platform
- Specialists with same certifications
- Geographically nearby specialists

### Mobile Optimization

**Page Load Speed:**
- Target < 3 seconds on mobile (3G)
- Lazy load specialist cards
- Optimize image sizes
- Compress JSON responses

**Mobile Layout:**
- Single column layouts
- Large touch targets (48px minimum)
- Readable font sizes (16px minimum)
- Sufficient spacing between elements

**Mobile SEO:**
- Mobile-first indexing ready
- Viewport meta tag configured
- Touch-friendly navigation
- Fast Core Web Vitals

## Structured Data Strategy

### Person Schema (Specialist Profiles)

```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "[Specialist Name]",
  "description": "[Brief bio]",
  "image": "[Avatar URL]",
  "jobTitle": "[Title]",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "[Rating]",
    "ratingCount": "[Count]"
  },
  "knowsAbout": ["[Platform]", "[Niche]"],
  "knowsLanguage": ["[Language 1]", "[Language 2]"],
  "workLocation": {
    "@type": "City",
    "name": "[Location]"
  }
}
```

### AggregateOffer Schema (Pricing)

```json
{
  "@context": "https://schema.org",
  "@type": "AggregateOffer",
  "priceCurrency": "USD",
  "lowPrice": "[Lowest price]",
  "highPrice": "[Highest price]",
  "offerCount": "[Specialist count]"
}
```

## Monitoring & Analytics Setup

### Recommended Metrics

**Search Console:**
- Impressions by page type
- Click-through rate (CTR) by page
- Average position by keyword
- Coverage and crawl errors
- Mobile usability issues

**Google Analytics:**
- Organic traffic by page
- Conversion rate by landing page
- Specialist profile engagement
- Click-through to contact forms
- Geographic distribution of traffic

**Keyword Tracking:**
- Monthly ranking changes
- Keyword trend analysis
- Competitor benchmarking
- Long-tail keyword performance

## Quarterly Content Review

**What to Review:**
1. Search Console performance data
2. Top performing keywords
3. Low-performing pages
4. New ranking opportunities
5. Competitor activity

**Optimization Actions:**
1. Update metadata for underperforming pages
2. Expand content for high-interest keywords
3. Add internal links to high-value pages
4. Refresh outdated specialist profiles
5. Add new structured data markup
