## 🛍️ Marketplace Pages

### 1. Main Marketplace Page: `/marketplace`

**Purpose:** Public specialist directory with filtering and search

**Features:**

#### Header
- Back to home link
- Page title + description
- Modern navigation

#### Featured Specialists Section
- Top 3 featured specialists displayed prominently
- Each card shows:
  - Avatar emoji
  - Name with verified badge
  - Hourly rate
  - Rating (e.g., 4.9★ with review count)
  - ROAS metric (e.g., 4.8x)
  - Key stats (campaigns, success rate, experience, response time)
  - Certifications (Meta, Google, TikTok, Yandex)
  - Specialties (E-commerce, SaaS, Lead Gen, Brand, Performance)
  - Location + quick contact button

#### Advanced Filtering Section
1. **Sorting Options**
   - By Rating (highest first)
   - By ROAS (highest first)
   - By Price (lowest first)

2. **Rating Filter**
   - All / 4.0+ / 4.5+ / 4.8+

3. **ROAS Filter**
   - All / 3x+ / 4x+ / 5x+

4. **Specialty Multi-Select**
   - E-commerce 🛍️
   - SaaS 💻
   - Lead Generation 📋
   - Brand Building ✨
   - Performance 📈

5. **Certification Multi-Select**
   - Meta Certified 📘
   - Google Partner 🔵
   - TikTok Certified 🎵
   - Yandex Expert 🔴

#### Results Display
- Real-time result count
- Grid layout (2 columns mobile, 3 columns tablet+)
- Empty state with helpful message
- Filterable results update instantly

#### Call-to-Action Section
- Heading: "O'zingiz markolog bo'lishni xohlaysizmi?"
- CTA: "Specialist sifatida ro'yxatdan o'tish"
- Links to /register with specialist role

---

### 2. Specialist Profile Page: `/marketplace/specialists/[id]`

**Purpose:** Detailed specialist information and booking

**Layout:**

#### Header Section
- Back to marketplace button
- Specialist avatar, name (with verified badge), title
- Sticky navigation for easy backlinks

#### Main Content (3-column grid)

**Left Column (2/3 width):**

1. **Statistics Card**
   - ROAS (e.g., 4.8x)
   - Rating with star (e.g., 4.9★)
   - Success rate (e.g., 94%)
   - Review count (e.g., 127)

2. **Three Content Tabs**

   **Tab 1: Overview (Default)**
   - Full bio/description
   - Background and methodology
   - Statistics subsections:
     - Years of experience
     - Campaigns managed
     - Total spend managed
     - Response time
   - Languages spoken
   - Platform certifications

   **Tab 2: Portfolio**
   - Case studies with:
     - Project title
     - Industry
     - ROAS achieved
     - Spend amount
   - Example:
     ```
     E-commerce Brand — Uzbek Market
     Fashion & Apparel
     6.2x ROAS | $125K spend
     ```

   **Tab 3: Testimonials**
   - Customer quotes with:
     - Rating (1-5 stars)
     - Quote text
     - Customer name and title
   - Example:
     ```
     ⭐⭐⭐⭐⭐
     "Dilshod bizning kampaniyani 3 oydada 2x ROAS o'sishiga 
     yordam berdi. Professionali va responsive."
     — Farida M., E-commerce CEO
     ```

**Right Column (1/3 width):**

3. **Booking Sidebar (Sticky)**
   - Service price per hour
   - Response time
   - Location
   - Two CTAs:
     - "Xabar yuborish" (Message)
     - "Foydalanuvchi profili" (User profile)
   - Info box explaining Performa payment system

## 🔄 User Flows

### Landing Page → Marketplace
```
Hero Section CTA
    ↓
"Markolog yoki agentni topish"
    ↓
/marketplace (specialist directory)
    ↓
Browse or filter specialists
    ↓
Click on card → /marketplace/specialists/[id]
```

### Sign Up as Specialist
```
Landing Page
    ↓
Marketplace CTA: "O'zingiz markolog bo'lishni xohlaysizmi?"
    ↓
/register?role=specialist
    ↓
Specialist onboarding flow
```
